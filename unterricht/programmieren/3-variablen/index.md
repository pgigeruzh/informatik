# Prüfungsvorbereitung

> Nennen sie die vier wichtigsten Datentypen und geben sie je ein Beispiel dazu.

> Was ist der Output von diesem Programm?
> ```python
> x = int(2.6)
> y = 2
> print(x * 2)
> ```

> Was ist der Output von diesem Programm?
> ```python
> x = "Hello "
> y = "World"
> print(x + y)
> ```

> Was ist der Output von diesem Programm?
> ```python
> a = 6
> b = 1.5
> c = a + b + (a / b)
> print("Output: " + str(c))
> ```

> Was macht dieses Programm?
> ```python
> x = float(input("Geben sie eine Zahl ein: "))
> print(x * x)
> ```
