Prüfungsvorbereitung

Was ist der Output von diesem Programm? Erstellen sie ein Flussdiagramm dazu.

x = "Hello"
y = "Tom"

if x != "Hello":
  y = "Tim"

else:
  y = "Tina"

print(y)

Was ist der Output von diesem Programm? Erstellen Sie ein Flussdiagramm dazu.

x = 5

if x < 0:
  print("a")

elif x > 0:
  if x == 5:
    print("b")

  print("c")

else:
  print("d")
