Prüfungsvorbereitung

Was ist der Output von diesem Programm?

def meine_funktion(x, y, z):
    if x > y:
        return z
    else:
        return z - 1

x = meine_funktion(1, 2, 3)
print(x)

Was macht diese Funktion (in Worte gefasst)?

def meine_funktion(x, y):
    if x > y:
        return x
    else:
        return y
